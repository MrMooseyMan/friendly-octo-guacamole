## 📱 Mobile Workflow

### Static Web Projects (HTML/JS/CSS)
1. Build in Cursor Mobile
2. Push to `main` branch
3. GitHub Pages auto-deploys
4. Open the Pages URL on your phone and play!

### Python/Node Projects
1. Open a [GitHub Codespace](https://github.com/codespaces)
2. Run the dev server (e.g., `python app.py`)
3. Copy the forwarded port URL
4. Open that URL on your phone

## ⚙️ Enable GitHub Pages (One-Time Setup)

1. Go to your repo on GitHub
2. Click **Settings** → **Pages**
3. Under "Source", select **Deploy from a branch**
4. Choose `main` branch, `/ (root)` folder
5. Click **Save**
6. Your site will be live at `https://mrmmooseyman.github.io/friendly-octo-guacamole/`
