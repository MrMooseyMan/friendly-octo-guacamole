# Firework Simulator

A simple, colorful firework simulator built with HTML5 Canvas and JavaScript.

## Features

- **Interactive**: Click or tap anywhere on the screen to launch a firework to that location.
- **Automatic**: Fireworks launch automatically at random intervals.
- **Physics**: Includes gravity, friction, and particle decay.
- **Visuals**: Colorful trails and explosions.

## How to Run

1. Open `index.html` in a modern web browser.
2. Enjoy the show!
